# Landing Page Components

This directory contains modular components for the TKIT College landing page, designed for guest users to explore the college's achievements, events, and activities.

## Components

### Core Components
- **GuestNavbar.js** - Navigation bar for guest users with smooth scrolling
- **HeroSection.js** - Main hero section with college statistics and call-to-action
- **AboutSection.js** - Mission, vision, and values of TKIT College
- **AchievementsSection.js** - Interactive cards showcasing college achievements
- **EventsSection.js** - Upcoming college events and festivals
- **ActivitiesSection.js** - Student clubs and activities
- **TestimonialsSection.js** - Auto-rotating testimonials from students and faculty
- **ContactSection.js** - Contact information and contact form
- **Footer.js** - Footer with links and social media

## Features

### Interactive Elements
- Smooth scrolling navigation
- Intersection Observer animations
- Auto-rotating testimonials carousel
- Hover effects on cards
- Responsive design for all screen sizes

### Content Sections
1. **Hero Section** - Welcome message with key statistics
2. **About Section** - College mission, vision, and values
3. **Achievements** - NAAC accreditation, awards, placement records
4. **Events** - Technical festivals, cultural events, sports week
5. **Activities** - Student clubs and societies
6. **Testimonials** - Student and faculty testimonials
7. **Contact** - Contact form and information

### Design Features
- Modern gradient backgrounds
- Card-based layouts
- Interactive hover effects
- Smooth animations
- Mobile-responsive design
- Professional color scheme (blue theme)

## Usage

The landing page is automatically displayed for non-authenticated users at the root route (`/`). Authenticated users are redirected to the dashboard.

## Styling

All components use Tailwind CSS for styling with:
- Responsive grid layouts
- Smooth transitions
- Professional color palette
- Modern typography
- Interactive elements
